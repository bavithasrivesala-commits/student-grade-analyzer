# student-grade-analyzer
Python console app that analyzes student marks and assigns grades.
Accepts marks for multiple students
Calculates average marks
Displays highest and lowest marks
Assigns grades using conditional logic
90 and above → A
75–89 → B
60–74 → C
50–59 → D
Below 50 → Fail
How to Run
1.Run the Python file
2.Enter the number of students
3.Enter marks for each student
4.The program will display statistics and grades
